# Installation Instructions

## Prerequisites
- Ensure you have the following installed:
  - Git
  - Node.js (version XX or later)
  - NPM (Node Package Manager)

## Clone the Repository
1. Open your terminal.
2. Run the following command:
   ```bash
   git clone https://github.com/talicdavor/stremio-kodi-subtitle-server.git
   ```
3. Navigate into the project directory:
   ```bash
   cd stremio-kodi-subtitle-server
   ```

## Install Dependencies
1. Use npm to install the necessary dependencies:
   ```bash
   npm install
   ```

## Configuration
- Create a `.env` file based on the example provided (if applicable) and modify it according to your settings.

## Running the Server
1. Start the server using:
   ```bash
   npm start
   ```
2. You can access the server at `http://localhost:PORT` (replace `PORT` with the configured port, default is often 3000).

---

Make sure to keep your dependencies up to date by occasionally running `npm update`.
