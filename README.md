# Stremio Kodi Subtitle Server

## Overview
The Stremio Kodi Subtitle Server is a powerful tool that enables Kodi users to leverage Stremio's subtitle capabilities seamlessly within their Kodi environment. This project aims to provide users with a simple and effective way to integrate subtitle functionalities, enhancing their media consumption experience.

## Features
- **Automatic Subtitle Downloading**: Automatically fetches subtitles for movies and TV shows from various sources.
- **Language Support**: Supports multiple languages, providing subtitles according to user preferences.
- **Customizable Settings**: Users can customize settings to match their viewing habits.
- **Integration with Kodi**: Easy integration with Kodi for a smooth user experience.

## Installation
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/talicdavor/stremio-kodi-subtitle-server.git
   cd stremio-kodi-subtitle-server
   ```
2. **Install Dependencies**:
   Make sure to have Node.js and npm installed, then run:
   ```bash
   npm install
   ```
3. **Run the Server**:
   ```bash
   npm start
   ```

## Usage
1. Open Kodi and navigate to the subtitle settings.
2. Add the Stremio Kodi Subtitle Server as a subtitle provider.
3. Enjoy enhanced subtitle capabilities while watching your favorite shows!
